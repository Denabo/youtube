# 🎬 Multi-Platform Short Video Pipeline

Проект для **потоковой (пакетной) сборки коротких видео** под:
- YouTube Shorts
- TikTok
- Instagram Reels
- VK Клипы

Поддерживается:
- наложение фона;
- наложение баннера (с хромакеем);
- фоновая музыка;
- автосубтитры через Whisper;
- пакетная очередь обработки;
- отдельная утилита нарезки по тайм-кодам.

---

## 📁 Актуальная структура

```text
youtube/
├── main.py                # Точка входа (терминальный UI)
├── terminal_ui.py         # Меню очереди и настроек обработки
├── batch_processor.py     # Пакетная обработка + рендер под платформы
├── subtitles.py           # Генерация и отрисовка субтитров
├── chroma_key.py          # Удаление зелёного фона у баннера
├── video_cutter.py        # Нарезка видео по списку тайм-кодов
├── config.py              # Вся конфигурация пайплайна
├── main_scripts/
│   └── Obrezka.py         # Совместимость, прокси к video_cutter.py
├── input/
│   ├── clips/             # Исходные клипы
│   ├── music/             # Музыка
│   ├── backgrounds/       # Фоны
│   └── banners/           # Баннеры
├── output/
│   ├── youtube/
│   ├── tiktok/
│   ├── instagram/
│   └── vk/
└── processed/             # Уже обработанные клипы
```

---

## ⚙️ Установка

```bash
pip install moviepy openai-whisper pillow numpy scipy
```

И установите ffmpeg в систему.

---

## 🚀 Быстрый старт

1. Положите материалы в папки `input/*`.
2. Запустите интерфейс:

```bash
python main.py
```

3. В меню:
- создайте очередь;
- выберите режим сборки;
- выберите платформы (`все` или конкретные);
- запустите обработку.

Результаты попадут в `output/<platform>/`.

---

## 🧠 Режимы обработки

Настраиваются в `config.py` (`PROCESSING_MODES`):
- обрезка под 9:16;
- фон + уменьшенный клип;
- баннер поверх;
- универсальный режим (комбинируется флагами).

---

## 🌍 Платформенные профили

В `config.py` (`PLATFORM_PROFILES`) задаются:
- размер кадра;
- fps;
- отображаемое имя платформы.

Можно добавить свой профиль и использовать его в очереди.

## ✂️ Нарезка по списку тайм-кодов

### Вариант A — список в файле

Создайте `cuts.txt` (первая строка `start,end,name` — опционально):

```text
start,end,name
00:00:05,00:00:15,intro
00:00:20,00:00:35,hook
00:01:00,00:01:20,
```

Запуск:

```bash
python video_cutter.py --input input/clips/source.mp4 --output output/cuts --ranges-file cuts.txt
```

### Вариант B — список в коде

Отредактируйте `CUT_RANGES` в `video_cutter.py` и запустите без `--ranges-file`.

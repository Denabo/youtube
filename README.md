# 🎬 YouTube Shorts Video Creator

Автоматическая система для создания профессиональных YouTube Shorts видео с субтитрами, баннером и музыкой.

## 📁 Структура проекта

```
youtue/
├── config.py                 # Конфигурация проекта
├── logger.py                 # Система логирования
├── chroma_key.py            # Удаление зелёного фона
├── background_maker.py      # Создание фона с баннером
├── subtitles.py             # Генерация субтитров (Whisper)
├── video_composer.py        # Сборка финального видео
├── main.py                  # Главный скрипт
│
├── input/
│   ├── Clips/
│   │   └── clip.mp4         # Основное видео
│   ├── backgraund/
│   │   └── blurred_background.mp4
│   ├── banner/
│   │   └── banner_fixed.mp4
│   └── music/
│       └── music.mp3
│
└── output/
    ├── ready_background.mp4  # Готовый фон (создаётся автоматически)
    └── shorts_final.mp4      # Финальное видео
```

## 🚀 Установка

### 1. Установите Python 3.8+

### 2. Установите зависимости:

```bash
pip install moviepy openai-whisper pillow numpy scipy
```

### 3. Установите FFmpeg:
- Windows: скачайте с [ffmpeg.org](https://ffmpeg.org/download.html)
- Linux: `sudo apt install ffmpeg`
- Mac: `brew install ffmpeg`

## 📝 Настройка

### Откройте `config.py` и настройте пути:

```python
BASE_DIR = r"C:\Users\ваш_пользователь\PycharmProject\youtue"
```

### Настройте параметры:

```python
# Размер видео (формат Shorts)
SHORTS_WIDTH = 1080
SHORTS_HEIGHT = 1920

# Громкость
MUSIC_VOLUME = 0.2      # Музыка тише
VOICE_VOLUME = 1.0      # Голос громче

# Хромакей (удаление зелёного фона)
CHROMA_TOLERANCE = 100  # Чем выше, тем больше зелени удаляется
```

## 🎯 Использование

### Вариант 1: Полный процесс (первый запуск)

```bash
python main.py
```

Это создаст:
1. Готовый фон с баннером (`ready_background.mp4`)
2. Финальное видео с субтитрами (`shorts_final.mp4`)

### Вариант 2: Быстрый режим (фон уже готов)

```bash
python main.py --quick
```

Пропускает создание фона, сразу создаёт видео.

### Вариант 3: Только фон

```bash
python background_maker.py
```

Создаёт готовый фон без сборки видео.

### Вариант 4: Только видео

```bash
python video_composer.py
```

Собирает видео из готового фона.

## 🔧 Модули

### `config.py`
Все настройки проекта: пути, размеры, параметры.

### `logger.py`
Система логирования с таймерами выполнения.

### `chroma_key.py`
**Улучшенный хромакей:**
- Удаляет зелёный фон
- Убирает зелёный оттенок с краёв
- Сглаживает края маски

### `background_maker.py`
**Создаёт готовый фон:**
- Обрезает фон под формат 9:16
- Применяет хромакей к баннеру
- Зацикливает баннер
- Сохраняет готовый фон

### `subtitles.py`
**Генерирует субтитры:**
- Использует Whisper AI
- Разбивает на короткие фразы (по 2 слова)
- Добавляет стильное оформление

### `video_composer.py`
**Собирает финальное видео:**
- Накладывает клип на готовый фон
- Позиционирует клип чуть выше центра
- Добавляет музыку
- Добавляет субтитры

### `main.py`
Объединяет все модули в единый процесс.

## 🎛️ Тонкая настройка

### Позиция клипа

В `config.py`:

```python
CLIP_VERTICAL_POSITION = 0.35  # 0.5 = центр, 0.35 = выше центра
```

### Качество хромакея

```python
CHROMA_TOLERANCE = 100    # 80-120 оптимально
CHROMA_EDGE_BLUR = 2      # Размытие краёв
```

### Размер баннера

```python
BANNER_WIDTH_RATIO = 0.9  # 0.9 = 90% ширины экрана
BANNER_TOP_OFFSET = 0.05  # Отступ сверху
```

### Субтитры

```python
FONT_SIZE = 60                      # Размер шрифта
SUBTITLE_WORDS_PER_PHRASE = 2       # Слов в одной фразе
SUBTITLE_BG_COLOR = (0, 0, 0, 180)  # Фон субтитров
```

## 🐛 Решение проблем

### Зелёные края на баннере
Увеличьте `CHROMA_TOLERANCE` в `config.py`:
```python
CHROMA_TOLERANCE = 120  # Было 100
```

### Клип не в том месте
Измените `CLIP_VERTICAL_POSITION`:
```python
CLIP_VERTICAL_POSITION = 0.4  # Ниже
CLIP_VERTICAL_POSITION = 0.3  # Выше
```

### Субтитры не видны
Увеличьте размер или измените цвет:
```python
FONT_SIZE = 70
SUBTITLE_TEXT_COLOR = "yellow"
```

### Музыка слишком громкая/тихая
```python
MUSIC_VOLUME = 0.1  # Тише
MUSIC_VOLUME = 0.3  # Громче
```
